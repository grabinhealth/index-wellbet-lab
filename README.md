# index-wellbet-lab

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名、网站或服务，仅作为前端页面资源的集合与备份。

## 项目简介

`index-wellbet-lab` 主要用于存放和展示各类独立开发的 HTML 页面。  
这些页面可以是静态演示、实验性项目、临时工具页面或学习笔记的在线版本。  
仓库结构简单，方便快速浏览与部署。

## 目录说明

```
.
├── index.html          # 仓库主页（可选，用于展示页面列表）
├── pages/              # 归档页面文件夹
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── README.md           # 本文件
```

- `pages/` 目录下存放所有归档的 HTML 页面，每个文件为一个独立页面。
- 页面命名建议使用有意义的英文或拼音，便于识别。

## 页面归档说明

- 每个 HTML 文件应保持独立，不依赖本仓库外的资源（外部 CDN 除外）。
- 页面内可包含 CSS、JavaScript 或图片（建议使用 base64 或相对路径）。
- 归档的页面内容不限制主题，但应符合开源社区基本规范。

## 维护说明

- 欢迎提交 PR 添加新的归档页面。
- 请确保新增页面不包含恶意代码、广告弹窗或侵犯他人权益的内容。
- 仓库维护者保留对页面内容进行审核或删除的权利。
- 若需修改已有页面，请直接更新对应文件并更新备注。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开 `pages/` 下的任一 HTML 文件即可查看。
3. 也可通过 GitHub Pages 部署后在线访问（需自行配置）。
